## Crew · Auditor

This repo has an **auditor** desk — a read-only reviewer of this repo's canon (in
`canon/`). When the canon needs checking for contradictions, stale claims, schema violations, or `canonical` /
`verified` pages that aren't actually supported, **delegate to the `auditor` agent** rather than
eyeballing it yourself. The Auditor only reports findings (page + tier); fixes still go through
capture → compile → review like any other change — never a direct edit.
